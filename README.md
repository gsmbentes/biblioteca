# Sistema de Biblioteca em Python

## Sobre o projeto
Este projeto é um sistema simples de gerenciamento de biblioteca desenvolvido em Python que funciona diretamente pelo terminal. O objetivo é simular as operações básicas de um balcão de atendimento de biblioteca, como o cadastro de acervo, registro de usuários, empréstimos e devoluções, aplicando conceitos de Programação Orientada a Objetos (POO) e validação de dados.

## Funcionalidades
O sistema opera através de um menu interativo no terminal e permite:
* Cadastrar livros no catálogo.
* Cadastrar novos usuários.
* Pegar um livro emprestado (alterando sua disponibilidade).
* Devolver um livro ao acervo.
* Sair do sistema com segurança.

## Tecnologias utilizadas
* **Python 3.13.2** (utilizando apenas a biblioteca padrão, sem dependências externas).
